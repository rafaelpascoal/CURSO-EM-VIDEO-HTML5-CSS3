Para que possamos incluir arquivos para download em páginas, incluímos o comando type="" e escolhemos o código que será colocar entre as aspas em www.iana.org (no caso de PDF é o código application/pdf e no caso de arquivos zip é o código application/zip).

Com relação ao arquivo pdf, não deu certo, pois o sistema reconheceu o arquivo como um arquivo já baixado no computador, mas funciona. Já o arquivo zip funcionou, pois era a primeira vez que era baixado.

Outro comando utilizado é o download="Aqui, coloca o nome do arquivo"
