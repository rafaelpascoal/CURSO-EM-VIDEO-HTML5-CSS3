Como seleciona e tirar seleção de um radio?

Simplesmente, mantenha todos os itens de radio com o mesmo name. Dessa forma, o objeto que selecionar será o escolhido e, se quiser alterar a escolha, é só selecionar a outra opção escolhida que a anterior será desmarcada. Esse recurso serve apenas para RADIO.

CHECKED

É o recurso utilizado para deixarmos uma opção pré selecionada. Esse recurso serve tanto para RADIO quanto para CHECKBOX.

VALUE

Informamos o VALUE no campo SEXO, pois a URL retorna a informação apenas que sexo está ON. Com a inclusão de VALUE, a URL informará qual sexo foi selecionado.
