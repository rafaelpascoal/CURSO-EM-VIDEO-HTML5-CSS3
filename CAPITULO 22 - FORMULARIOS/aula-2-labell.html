Label

AUTOCOMPLETE: OFF

Esse recurso (autocomplete:off) é utilizado para que nao haja sugestão de autocompletar o texto que está sendo digitado pelo usuário. Ele é incluído dentro da TAG FORM (dentro de <'form'>)

ACTION

O recurso ACTION aparece automaticamente no momento em que a TAG FORM é acionada. Ele é o endereço de destino do formulário (para onde os dados irão depois que o usuário apertar o botão de Enviar). No caso do curso, utilizamos a linguagem PHP (linguagem mais popular), mas existem outras: ASP, Java servidor, por exemplo.

NAME

O recurso NAME é mais útil para HTML e PHP e é utilizado para nomear o item incluído (a exemplo do item nome que recebeu o name de nome também).

ID

Já o ID é mais útil em Javascript, inclusive para o uso do recurso LABEL.

LABEL

O recurso LABEL é utilizado para realizar a ligação entre o texto incluído e o texto que será digitado na caixa de diálogo (Exemplo: O item Nome será ligado à caixa de diálogo que se encontra ao seu lado e o que for digitado nela será considerado um nome). Dentro do recurso label existe o recurso FOR. Esse recurso determina a ligação e o ID deverá ser utilizado para que essa ligação seja realizada.

Além dessa ligação ser importante, ela gera um aumento da área, tendo em vista que, independentemente do usuário clicar na caixa de diálogo ou no texto identificador (exemplos: Nome e Sobrenome), a caixa de diálogo será acionada. Isso é relevante para usuários que utilizam celulares, pois as telas são menores, o que dificulta a seleção de qualquer formulário e, com esse recurso habilitado, facilitará a referida seleção.
