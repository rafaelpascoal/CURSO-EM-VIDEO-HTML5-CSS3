TEXTO E SENHAS

INPUT: TEXT

O recurso INPUT possui alguns recursos, dentre eles, o TEXT que é utilizado para inserir textos (Nomes, sobrenomes, usuários, etc)

INPUT: PASSWORD

Já para senhas, utilizamos o input do tipo password.

REQUIRED

O recurso required é inserido dentro de input para que aquele campo seja obrigatório, não sendo possível enviar o formulário sem seu preenchimento.

MINLENGHT E MAXLENGHT

São recursos que determinam o mínimo e máximo de caracteres exigidos para que seja validada a informação inserida nos campos (Exemplo: Para o usuário, é exigido o minimo de 5 caracteres e máximo de 15 caracteres. Se for menos que 5 ou mais que 15, não será validado e o usuário terá que corrigir para prosseguir).

SIZE

Size é um recurso que configura o tamanho da caixa de diálogo.

PLACEHOLDER

Utilizamos o recurso do placeholder para inserirmos um texto de sugestão dentro da caixa de diálogo (Exemplo: Na caixa de diálogo de Usuário, inserimos o texto nome do usuário com o placeholder). Assim que o usuário selecionar a caixa e iniciar a digitação, o texto desaparecerá.
