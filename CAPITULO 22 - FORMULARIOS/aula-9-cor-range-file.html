INPUT:FILE

Nas ocasiões em que utilizamos a ferramenta INPUT:FILE, necessariamente deveremos utilizar o method:POST, e não o méthod:GET que é o padrão, tendo em vista que a inserção de arquivos é considerada sensível e, principalmente, os arquivos tendem a ter mais de 3 mil bites.

CONFIGURAÇÃO DO INPUT:RANGE

Pesquisando, descobri que o comando para retirar toda a configuração do referido input é -webkit:appearence: none.

Depois de retirar toda a configuração padrão do input, podem-se realizar configurações diversas, mas não consegui alterar o formato e cor da bola azul.

VALUE

Novamente, o recurso value é utilizado para determinar o que queremos dos inputs. No caso do input range, conseguimos determinar em qual número a bola azul iniciará (se no meio, no final ou no início). Já no input color conseguimos determinar qual cor será apresentada inicialmente (verde, azul, vermelha, etc).
