Citações sem quebra de linhas

Como diria o pai de um amigo, o computador é um burro muito rápido. Para darmos sentido a uma citação, fazendo referência a algo que alguém disse, utilizamos a tag Q (quotation).

Citações completas com quebras de linhas

Para citações completas ou maiores, utilizamos a tag BLOCKQUOTE. Nela, podemos incluir o endereço eletrônico de origem, adicionando cite (no Visual Code coloquei o link do curso que estava fazendo apenas para fins de exercício).
