Modelos de Caixas

Box Level - Costumam existir independentemente, como caixas mesmo.

Inline level - são itens que não quebram linhas. Exemplo:Temos os links com a tag a que não quebram a linha.

Lembrando que:

Para a borda (border), temos várias opções de formatação:

  • border
  • border-width
  • border-style
  • border-color

Sendo que, para cada um deles, podemos selecionar qual lado queremos alterar adicionando o comando top, bottom, left e right (Exemplo: border-left-color).

Podemos, também, adicioná-los aos códigos de forma simplificada, seguindo sempre a sequência mencionada na lista:

Exemplo: border: 5px groove blue;

No exemplo acima, simplificamos o código border, agrupando várias formatações ao mesmo tempo: width (largura), style (estilo) e color (cor). Essa simplificação é chamada de shorthand

Podemos utilizar a shorthand para vários comandos, desde que eles possuam identidade de formatação (border, padding, margin, etc). Apenas devemos sempre seguir os parâmetros e respeitar a ordem de comandos.

Para formatação de características não numéricas, seguir a sequência:

  • width
  • style
  • color

Para formatação de características numéricas, seguir a sequência:

  • top
  • right
  • bottom
  • left

Para lados (top, right, bottom, left) devemos sempre seguir o movimento horário para inclusão dos comandos.

Se quisermos que o box fique centralizado na página e que se adapte ao dispositivo e tamanho de tela utilizado, devemos incluir o comando margin: auto (para textos, devemos utilizar o comando text-align: center).


Por padrão, o H1 é display: block (box-level).


Contudo, se eu quero transformar ele em inline-level, preciso apenas incluir o comando display:inline; nos códigos de style.
