    2. Table Data - Dados de tabela - td

Configurações de Tabela - CSS

Border

O comando border é utilizado para criar uma borda entre as células (no caso, em td que são os dados relacionados às celulas).

Border-collapse: Collapse

O comando border collapse é utilizado para unir, fundir as bordas das células para que estas tenham o aspecto de tabelas de excel

Observação Final

Lembrando que, caso haja um desrespeito da hierarquia das tabelas (td contido por tr e ambos contidos por table), haverá inconsistência no resultado final da tabela.
