AGRUPAR COLUNAS

Para agrupar linhas (com efeito de seleção de linha - como no exemplo de Cláudio) é mais imples, pois criamos uma classe e configuramos a referida classe.

Já para criarmos o mesmo efeito em colunas é mais complexo, pois precisamos criar um comando (geralmente incluído após a tag table) chamado colgroup, incluindo todos os itens de colunas que queremos destacar (a exemplo das colunas nome, sexo, idade e profissão) e incluindo uma classe para cada um e, posteriormente, configurar em CSS.
