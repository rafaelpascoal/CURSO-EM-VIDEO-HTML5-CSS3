Formato de vídeos para sites

Para podermos alterar as extensões de vídeos, utilizamos o aplicativo HandBrake. Nele, podemos alterar para várias extensões.

Para que o vídeo tenha uma capa inicial (antes que o usuário aperte o play), incluímos o comando poster na tag vídeo para que a tela não fique pausada no próprio vídeo e traga uma sensação mais profissional para a mídia.

A ordem dos sources é importante, pois o navegador sempre tentará reproduzir do primeiro ao último arquivo incluído (no caso, seria a sequência: mp4, m4v, webm e ogv)

Lembrando que, todos os arquivos incluídos aqui são de hospedagem local (estão no meu computador).

Por fim, é importante frisar que, os vídeos hospedados dessa maneira (hospedagem local) gera um aumento de custos para o cliente. Não é a maneira mais eficaz e econômica de hospedagem, pois, se os arquivos são grandes, gerará um grande tráfego de dados.
