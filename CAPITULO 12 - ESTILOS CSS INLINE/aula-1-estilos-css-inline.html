Estilos CSS inline

O estilo CSS inline inclui as configurações em cada item adicionado em um site (dentro, na mesma linha do item). Esse método é utilizado em casos muito específicos em que o desenvolvedor precisa alterar um item, de forma isolada, não sendo utilizado para a configuração geral.
