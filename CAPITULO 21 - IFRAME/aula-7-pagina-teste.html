COMANDO SANDBOX - PROTEÇÃO 1

Meu Site Principal

Para manter os dados do usuário seguros com relação a qualquer tentativa maliciosa do site incluído no iframe, podemos introduzir o comando sandbox="sandbox" dentro do código do iframe.

Além da opção sandbox (segurança máxima), o comando SANDBOX possui as seguintes opções:

  • allow-same-origin - Para IFRAMES que possuem o mesmo servidor.
  • allow-forms - PAra autorização apenas de formulários;
  • allow-scripts - Para autorização apenas de scripts.

Com isso, o usuário, mesmo se quiser cadastrar alguma informação nos formulários disponibilizados no iframe, não conseguirão concretizar a solicitação (Haverá um bloqueio dos recursos e, ao clicar em "Cadastrar" no formulário dentro do iframe, este não funcionará).

COMANDO REFERRERPOLICY - PROTEÇÃO 2

Outro comando que traz mais segurança ao IFRAME é o REFERRERPOLICY que possui duas opções:

  • referrerpolicy="no-referrer-when-downgrade" - E o padrão do comando.
  • referrerpolicy="no-referrer" - Caso queira mais segurança - O item que está contido no IFRAME não coletará nenhum tipo de informação do usuário com esse comando.
