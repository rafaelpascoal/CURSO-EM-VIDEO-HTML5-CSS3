Configuração de IFRAME

Por padrão, um iframe possui tamanho de 300 x 150 px.

É possível alterar, tanto o width quanto o height nos iframes quanto em css (caso queira uma alteração pontual, vale a pena configurar no iframe. Caso seja uma alteração geral, compensa configurar em CSS).

Lembrando que, em toda configuração de tamanho e/ou características de iframe, as realizadas em CSS prevalecerão. Portanto, CSS é soberano.

SCROLLING

É o comando que determina se será apresentada ou não barra de rolagem.

O scrolling possui 3 tipos:

  • auto - Aparecerá a barra de rolagem automaticamente, se for necessário (Se o conteúdo exceder o tamanho do iframe).
  • yes - Aparecerá a barra de rolagem necessariamente.
  • no - Não aparecerá a barra de rolagem, independentemente de ter ou não mais conteúdo no site.

FRAMEBORDER

O comando frameborder é utilizado com as referências 0 e 1 para a borda do iframe (com 0 não terá borda e com 1 terá a borda)

Esse comando serve para que possa configurar a borda e criarmolduras.

SRC

O comando SRC nada mais é que o endereço de destino para o iframe, ou seja, qual será a página que será mostrada no iframe.
