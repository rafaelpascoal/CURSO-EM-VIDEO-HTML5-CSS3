Codigos para imagens

Utilizamos alguns comandos relacionados às imagens.

Background-image

O background-image é utilizado para introduzir imagens de fundo. Nele, há a possibilidade de introduzirmos uma imagem (url), introduzirmos cores misturadas (linear-gradient).

Background-size

O comando background-size controla o tamanho da imagem. No exemplo do ícone, introduzimos o tamanho 100px por 100px para que ele ficasse do tamanho que está na tela de demonstração.

Background-repeat

O comando background-repeat é utilizado tanto para repetir uma imagem pela tela toda (já é o padrão - repeat) como pode fazer repetir horizontalmente (repear-x), verticalmente (repeat-y) e até não repetir (no-repeat) como bem demonstrado na tela de exibição.
