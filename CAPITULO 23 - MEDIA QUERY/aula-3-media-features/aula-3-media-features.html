MÚLTIPLAS MEDIA FEATURES COM CSS (Aula 03) e SEGUINDO A ORIENTAÇÃO DO DISPOSITIVO (Aula 04)

As Media Types ou tipos de mídias, são aquelas demonstradas no exercício anterior (screen, print, impressora de braile, all, etc).

Já as MEDIAS FEATURES são as características de mídias, ou seja, sua dimensão e sua orientação [orientation - portrait (retrato) ou landscape (paisagem)] são exemplos de características.

Enquanto as MEDIA TYPES seriam os celulares, as MEDIA FEATURES seriam os vários tipos de celulares com suas variadas dimensões.

Por fim, MEDIA QUERIES seriam a junção de MEDIA TYPES e MEDIA FEATURES.
